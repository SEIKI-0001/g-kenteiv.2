# G検定クエスト

G検定の資格取得サポートサービスを、ゲーミフィケーション重視で再設計した別バージョンです。

## コンセプト

学習タスクを「クエスト」、弱点対策を「ダンジョン」、模擬試験を「ボス戦」として扱い、毎日の学習継続をXP、コイン、バッジ、カード収集で後押しします。

## ターゲット

- 学習継続が苦手で、進捗の手応えが欲しいG検定受験者
- 短い時間でも達成感を得ながらAI基礎を積み上げたい社会人・学生
- 暗記、演習、模試をゲームのようなサイクルで回したい初学者

## 主な画面

- ホーム: 今日のクエスト、XP、合格ロード、弱点ダンジョン
- クエスト: デイリー学習タスクと報酬
- 弱点ダンジョン: 苦手単元を階層化して攻略
- ボス模試: 本番形式の模試と再戦導線
- カード図鑑: 頻出用語の収集・強化
- ギルド: 学習チームや比較表示のモック

## ローカル確認

```bash
python3 -m http.server 4174
```

ブラウザで `http://127.0.0.1:4174/` を開いて確認できます。
