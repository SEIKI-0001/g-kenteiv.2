# G検定マスタリーラボ

G検定対策サービスのゲーミフィケーション版です。前回の「ゲーム風UI」ではなく、学習サービスで実際に効きやすいゲーミフィケーション要素に絞って再設計しています。

## 設計方針

- 近接目標: 合格という大目標を、今日終えられる学習契約へ分解
- 即時フィードバック: 正誤だけでなく、間違えた理由をその場で返す
- 有意味報酬: 単なるポイントや装飾ではなく、説明力、再現力、復習完了の証跡を報酬化
- 復習間隔: 忘れ始めるタイミングで短い復習キューを出す
- 自律性: 学習者が「検索練習、演習、説明、復習」から選べる余地を残す
- リカバリー: 連続記録だけでなく、休息後に戻ってくる行動も評価

## 画面

- ダッシュボード: 今日の学習ループ、ミッション、設計レンズ、マスタリー、復習キュー
- ミッション: 検索練習、演習、説明、復習に分けた行動設計
- 復習キュー: 今日、明日、3日後、7日後の再提示
- マスタリー: G検定の単元別到達度と集中領域の選択
- リワード: 学習成果と結び付いたバッジ設計
- コーチ: 集中領域に合わせた学習リズム
